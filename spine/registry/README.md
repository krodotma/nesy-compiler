# Registry Assets

This directory contains canonical registry and schema artifacts consolidated for the Spine repo.

- `specs/`  — subsystem, cagent, semops, skills schema
- `state/`  — constitutional registry state
- `mcp/`    — effects registry
- `runtime/` — snapshots (skills registry, root registry ndjson)

These are source-of-record artifacts used by Pluribus and Dialogos.
